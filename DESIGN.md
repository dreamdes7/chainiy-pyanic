# Design

## Theme
Ретро-аналоговая чайная лавка: плёночное зерно, фактура бумаги, letterpress-оттиск. Тёплый, тактильный, душевный — как винтажная чайная жестянка, а не цифровой стор. Светлый, но **закоммиченный** табачно-тёплый субстрат (не кремовый near-white), кремовые карточки как этикетки поверх.

## Color (OKLCH-derived)
- Substrate (фон): `#d9c39b` — табачно-манильный, осознанный тёплый тон (не near-white).
- Substrate-2 (футер/секции): `#cdb588`.
- Card: `#efe6cf` — кремовая «этикетка», на ней чисто читаются фото товара.
- Ink: `#211b12` (текст, контраст ≥9:1).
- Ink-2: `#4a4030` (вторичный текст, ≥6:1).
- Rust `#bd5a28` — главный акцент, несёт характер бренда.
- Moss `#4c6138` — вторичный зелёный, основная кнопка.
- Stamp `#ad3526` — распродажа/удаление/ошибки.
- Line: `rgba(33,27,18,.20)`.

## Typography
- **Display / заголовки / лейблы / цены:** "Bitter" (слэб-сериф, кириллица) — letterpress, тин-лейбл. НЕ mono.
- **Body / UI / описания / таблицы:** "Golos Text" (тёплый гротеск, кириллица) — читабельность.
- Пара по контрастной оси (слэб-сериф + гротеск). Ни один не из reflex-reject списка.
- clamp() для заголовков, шаг ≥1.25, `text-wrap: balance` на h1–h3, длина строки 65–75ch.

## Tactility (ядро вайба — сохраняется)
- Плёночное зерно (анимированный SVG-noise) + виньетка поверх всего, в меру.
- Фактура бумаги на фоне.
- «Напечатанные» фото: тиснёная рамка + зерно поверх + аналоговая цветокоррекция.
- Letterpress-кнопки с жёсткой тенью-оттиском.

## Layout
- Ритм важнее сетки: featured — одна крупная «обложка» товара + сетка остальных (не монотонная решётка).
- Секции: крупный слэб-заголовок + тонкая линейка. **Без нумерации 01/02 и без eyebrow над каждой секцией.**
- Сетки товаров: `repeat(auto-fill, minmax(280px,1fr))`.
- Анимируем только transform/opacity (никаких width/padding).

## Bans (самоконтроль)
mono-лейблы · кремовый near-white фон · нумерация секций · eyebrow-рефлекс · side-stripe бордеры · градиентный текст · glassmorphism · одинаковые карточки-плитки.
